QBox NFC Tagger

Read and write NFC tags on QIDI Box filament spools

What It Does

QBox NFC Tagger connects to an Arduino Nano 33 BLE + PN532 NFC reader over Bluetooth to read and write filament data on MIFARE Classic 1K tags used in QIDI Box 3D printer spools. Identify material type, color, and manufacturer at a glance, or reprogram tags with new filament information.

Features

Bluetooth Low Energy scanner • One-tap tag reading • Filament material and color picker with color swatches • Tag writing with verification • Continuous read/write mode for hands-free operation • Demo mode for testing without hardware • Support for 20+ filament materials and colors

Requirements

iPhone running iOS 17 or later, an Arduino Nano 33 BLE microcontroller with PN532 NFC module running the NFC bridge firmware, and MIFARE Classic 1K NFC tags (FM11RF08S) as used in QIDI Box filament spools.
